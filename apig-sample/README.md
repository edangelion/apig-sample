# API Server

Simple Rest API using gin(framework) & gorm(orm)

## Endpoint list

### Emails Resource

```
GET    /api/emails
GET    /api/emails/:id
POST   /api/emails
PUT    /api/emails/:id
DELETE /api/emails/:id
```

### Users Resource

```
GET    /api/users
GET    /api/users/:id
POST   /api/users
PUT    /api/users/:id
DELETE /api/users/:id
```

server runs at http://localhost:8080
